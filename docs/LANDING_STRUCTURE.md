# OpenClaw Monitor Lite 首版 Landing 结构

## 目标
把首页进一步从“dashboard 展示页”推进为“产品首页”。

---

## 首版 Landing 最该讲的 6 件事
### 1. 这是给谁的
给已经在真实运行 OpenClaw 的独立开发者和小团队。

### 2. 第一版只卖什么
- 成本可见
- 告警可达
- 节点健康可见
- 错误日志可追

### 3. 为什么现在值得接入
当 cron、session、节点、成本和异常已经开始变复杂时，就值得接入。

### 4. 为什么它比通用 observability 更轻更贴 OpenClaw
强调 OpenClaw-native 运行视图，而不是通用 trace 平台。

### 5. 它和 security 模块的关系
security 负责首单体检；monitor 负责长期运行层。

### 6. 用户下一步怎么做
先试运行当前面板 → 再逐步收敛成长期 Agent Ops Lite 主壳。
