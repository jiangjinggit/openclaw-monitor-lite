# OpenClaw Monitor Lite 下一步动作

## 当前首页最该引导的动作
首页不该让用户一上来就理解所有面板细节，而应该先引导 3 个动作：

1. **先看当前运行视图**
2. **先判断自己是否已经进入 Agent Ops Lite 阶段**
3. **再决定是否要把这层作为长期主壳来接入**

---

## 为什么这样设计
- `monitor-lite` 不是短期首单产品
- 它更适合承接已经开始真实运行的用户
- 所以 CTA 更适合引导“理解自己是否到了这个阶段”，而不是直接卖复杂功能

---

## 当前最合理的首页动作顺序
1. 看定位
2. 看四个核心卖点
3. 看适用阶段
4. 看当前真实运行视图
5. 再理解后续升级空间
