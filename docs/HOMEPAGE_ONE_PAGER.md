# OpenClaw Monitor Lite 首页一页式收口

## 首页最该优先讲的 4 件事
1. 这是给已经在真实运行 OpenClaw 的人准备的。
2. 第一版只卖：成本、告警、节点健康、错误日志。
3. 它不是短期首单，而是长期运行层主壳。
4. 当前最重要的动作不是理解所有功能，而是先看清运行状态。

---

## 首页前半屏原则
- 先讲定位
- 再讲适合谁
- 再讲为什么现在值得接入
- 再讲 4 个核心卖点
- 之后才进入 dashboard 细节

---

## 收口原则
不要让首页一开始就像工程面板；
要先让用户理解这是一个轻量 Agent Ops Lite 主壳。
